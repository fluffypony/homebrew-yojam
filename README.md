# homebrew-yojam

Homebrew tap for [Yojam](https://yoj.am), a macOS browser picker and link router. The app itself lives at [fluffypony/yojam](https://github.com/fluffypony/yojam). This repo is just the cask.

## Install

    brew tap fluffypony/yojam
    brew install --cask yojam

## Uninstall

    brew uninstall --cask yojam

Add `--zap` to wipe preferences, caches, rules, and browser extension manifests as well:

    brew uninstall --cask --zap yojam

## Issues

File them at [fluffypony/yojam](https://github.com/fluffypony/yojam/issues), not here. This repo only changes when a new version ships.
